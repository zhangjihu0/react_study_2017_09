## 1. 安装bower 
bower 是一个前端框架的管理工具 jq angular bootstrap
npm node 包管理器 mime gulp
bower 是一个全局的命令行工具， 需要通过npm 全局安装；
npm 在安装node的时候自动安装的
```
npm install bower -g
```
## 2.安装react babel
```
bower install react babel
```
react 是我们需要的安装模块；
babel react的编译工具；
>bower install 之后会在当前项目下增加一个bower_components的文件夹。里面装着我们安装的模块
